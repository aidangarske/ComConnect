# Frontend

React app with Chakra UI.

## Start

```bash
npm install
npm run dev
```

Runs on `http://localhost:5173`

## Pages

- `/` - Home page
- `/login` - Login
- `/register` - Register
- `/dashboard-seeker` - Seeker dashboard
- `/dashboard-provider` - Provider dashboard
- `/profile` - User profile
- `/messages` - Messages
- `/admin` - Admin dashboard

## Features

- Role-based auth (provider, seeker, admin)
- Job posting and browsing
- User profile management
- Messaging interface
